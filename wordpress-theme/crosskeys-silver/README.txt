Crosskeys Silver Band — WordPress Theme
=======================================

Before uploading:
1. Copy Media/logo.png            → images/logo.png
2. Copy Media/<hero-image>.jpg    → images/hero-bg.jpg
3. Copy Media/conductor.jpg       → images/conductor.jpg
4. Copy Policies/*.pdf            → policies/*.pdf  (keep original filenames)

Upload to: /public_html/wp-content/themes/crosskeys-silver/

Then in WordPress admin:
- Appearance → Themes → Activate "Crosskeys Silver Band"
- Pages → Add New → "Home" (blank content) → Publish
- Settings → Reading → Static page → Homepage: Home
- Pages → Add New → "Policies" (slug must be "policies") → Publish
- Install Contact Form 7 + WP Mail SMTP for real email
- Vacancies → Add New to manage band vacancies

See DEPLOYMENT_GUIDE.md for full instructions.
